# Фотографии товаров — структура и правила

## Структура папок

Каждая категория — отдельная папка:

```
images/products/
├── armchairs/      Кресла
├── sofas/          Диваны
├── beds/           Кровати
├── dining/         Обеденные группы
├── storage/        Шкафы и хранение
├── terraces/       Террасы
├── loungers/       Лежаки и кушетки
├── pavilions/      Беседки
├── swings/         Качели и гамаки
├── pillows/        Подушки и пледы
└── rugs/           Дорожки и ковры
```

## Соглашение об именах файлов

Формат: `{slug-товара}-{номер}.jpg`

- **slug-товара** — короткое латинское имя (без пробелов и кириллицы)
- **номер** — порядковый номер фото (`1` главное, `2`–`5` дополнительные)

Примеры:
```
images/products/armchairs/gracia-1.jpg     ← главное фото кресла «Грация»
images/products/armchairs/gracia-2.jpg     ← дополнительное фото
images/products/armchairs/gracia-3.jpg     ← вид сзади
images/products/sofas/oblako-1.jpg         ← главное фото дивана «Облако»
images/products/terraces/laguna-1.jpg
images/products/terraces/laguna-2.jpg
```

## Технические требования

| Параметр | Значение | Зачем |
|---|---|---|
| **Формат** | JPG (или WebP) | Меньше веса, быстрее загрузка |
| **Размер** | 1400×1750 px минимум (соотношение 4:5) | Резкость на retina-дисплеях |
| **Вес файла** | до 400 KB | Быстрая загрузка страницы |
| **Цветовой профиль** | sRGB | Корректные цвета в браузере |

## Как сжимать фото

Если фото с камеры (3–8 MB каждое):

**На Mac:**
1. Открой в Preview
2. Tools → Adjust Size → 1400 пикселей по большей стороне
3. File → Export → JPEG quality 70-80%

**Через онлайн-сервисы:**
- [Squoosh.app](https://squoosh.app) — простой, от Google
- [TinyPNG.com](https://tinypng.com) — пакетная обработка

## Сколько фото на товар

| Тип товара | Минимум | Рекомендуется |
|---|---|---|
| Карточки в каталоге | 1 | 2 (для hover-эффекта) |
| Страница товара | 2 | 3–5 |

Главное фото (`*-1.jpg`) показывается в карточке и в галерее первым. Делай его самым удачным.

## Что показывать на фото

1. **Главное фото** — товар целиком, на нейтральном фоне (белый/серый/паркет)
2. **Деталь** — крупный план фактуры, соединения, фурнитуры
3. **В интерьере** — товар в обстановке, чтобы покупатель понял масштаб
4. **Альтернативный ракурс** — вид сбоку, сзади
5. **Размер** — товар с человеком/предметом для масштаба (опционально)
